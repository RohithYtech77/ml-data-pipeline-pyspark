# ML-Ready Data Pipeline — PySpark, AWS, Snowflake

Built scalable data pipelines using PySpark to prepare and deliver
clean, structured data for machine learning and analytics teams.

## Tech Stack
- Python, PySpark, pandas
- AWS S3, AWS Glue, AWS Athena
- Snowflake — serving layer
- Apache Airflow — orchestration
- Docker — local development

## Pipeline Flow
1. Ingest raw data from multiple sources (APIs, CSV, JSON)
2. PySpark transformations — cleaning, deduplication, feature engineering
3. Data Vault 2.0 modeling in Snowflake
4. Mart tables served to ML and analytics teams
5. Automated quality checks before each load

## Key Highlights
- Replaced 100% of manual data preparation workflows
- Supports ML forecasting pipelines with 85%+ accuracy
- Star schema and Data Vault 2.0 design patterns
- RBAC and encryption standards enforced
